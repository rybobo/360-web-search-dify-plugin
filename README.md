  - 智能问答：获取最新信息
  - 信息检索：快速查找中文内容

## 使用方法

  ### 1. 获取API密钥
  1. 访问 https://ai.360.com/platform
  2. 进入「开放平台」→「API Keys」
  3. 创建应用并复制密钥

  ### 2. 在Dify中配置
  1. 安装插件
  2. 填写API密钥
  3. 开始使用
